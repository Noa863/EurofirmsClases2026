## Descripción

Este proyecto consiste en un **ejercicio visual diseñado para entender cómo funcionan los selectores CSS** dentro de una estructura HTML.

Para facilitar la comprensión se utiliza una metáfora sencilla:

🏠 **Una casa**

donde existen personajes:

- fuera de la casa
- dentro de la casa
- dentro de habitaciones
- con características especiales

Esto permite visualizar claramente conceptos fundamentales de CSS como:

- selectores de etiqueta
- selectores de clase
- hijos directos
- descendientes

## Objetivos de aprendizaje

Al finalizar esta práctica el estudiante debería ser capaz de:

1. Construir una estructura HTML con elementos anidados.
2. Comprender la relación **padre → hijo → descendiente**.
3. Aplicar selectores CSS básicos.
4. Diferenciar visualmente elementos similares (`<p>`) dependiendo de su posición.
5. Entender cómo afecta la estructura HTML al comportamiento del CSS.

---

# 📂 Estructura del ejercicio

El ejercicio requiere exactamente la siguiente estructura:

- Dos `<p>` fuera del `<div>`
- Un `<div>` que representa la casa
- Dentro del `<div>`:
  - un `<p>` normal
  - un `<p>` con clase
- Un `<section>` dentro del `<div>`
- Dentro del `<section>` un `<p>`

# 📂 Estructura del ejercicio

El ejercicio requiere exactamente la siguiente estructura:

- Dos `<p>` fuera del `<div>`
- Un `<div>` que representa la casa
- Dentro del `<div>`:

### Fragmento del HTML

```html
<p>Niña fuera de la casa</p>
<p>Niño fuera de la casa</p>

<div class="casa">

  <p>Niña en la sala</p>

  <p class="especial">
    Niña especial
  </p>

  <section>
    <p>
      Niño dentro de la habitación
    </p>
  </section>

</div>
